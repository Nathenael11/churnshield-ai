 ChurnShield AI

AI-powered customer churn prediction system for telecom and financial services.

 Live Demo

[https://huggingface.co/spaces/nathe11/churnshield](https://huggingface.co/spaces/nathe11/churnshield)

 Model Performance

- AUC-ROC: 91.5%
- Accuracy: 88.3%
- Precision: 85.2%
- Recall: 84.7%

 Business Impact

- 20% churn reduction → $6.98 million annual savings
- ROI: 698%
- 9,912 customers retained

 Tech Stack

- Python (Pandas, NumPy, Scikit-learn)
- Jupyter Notebook
- Streamlit
- Hugging Face Spaces
